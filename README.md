# Assignment 01 - University Management System

## Course Information
**Course:** Object-Oriented Programming Lab (Java)

## Overview
This project implements a simple University Management System using Object-Oriented Programming (OOP) concepts in Java. The system manages different types of university personnel through a common framework while allowing access to specialized information when required.

## OOP Concepts Demonstrated
- Abstract Class
- Multilevel Inheritance
- Runtime Polymorphism (Dynamic Binding)
- Safe Downcasting using `instanceof`

## Class Hierarchy

UniversityMember
│
└── Employee
│
├── AcademicStaff
│ └── Professor
│
└── AdministrativeOfficer

## Features
- Store university personnel information.
- Display basic member details.
- Demonstrate different duties using runtime polymorphism.
- Access professor-specific research information through downcasting.
- Easy to extend with additional employee types.
